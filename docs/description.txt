遊戲論壇專題說明：

本專案旨在開發一款結合論壇與商城功能的遊戲交流平台，類似於巴哈姆特，提供玩家交流討論與遊戲商品交易的空間。我們將採用 ASP.NET Core MVC 作為後端框架，前端則使用 Bootstrap、HTML、CSS、JavaScript 進行開發，並以 MSSQL 作為資料庫。

===

本平台主要包含以下功能模組：

論壇系統：包含首頁、看板頁面、討論區文章列表與詳細頁面，支援文章發表、回覆、評價、訂閱與分類管理，並提供聊天室功能（待開發）。

新聞區：提供最新遊戲新聞與相關報導，包含留言板與管理者編輯功能。

個人頁面：用戶可管理個人日誌、討論區文章與收藏內容，並可與其他玩家互動。

商城系統：內建購物車、商品評價、結帳與訂單紀錄等功能，並提供排行榜與廣告輪播圖以提升使用者體驗。

會員與客服系統：支援帳號註冊、登入、社群登入、偏好設定，以及客服中心以解決使用者問題。

===

本專案的目標是打造一個完善的遊戲社群，讓玩家能夠即時交流、掌握最新資訊，並方便地購買遊戲相關商品，提升整體使用體驗。
